# Spring API Common Template

Spring Boot 기반 API 서버 개발 시 반복적으로 필요한 공통 기능을 미리 구성한 템플릿 프로젝트입니다.

## Features

- Common API Response
- Global Exception Handler
- Validation
- Swagger(OpenAPI)
- MyBatis
- Actuator
- Profile Management
- Request Logging
- Docker Support

## Tech Stack

- Java 21
- Spring Boot 3.5
- Gradle
- MyBatis
- MariaDB
- Lombok
- Swagger(OpenAPI)
